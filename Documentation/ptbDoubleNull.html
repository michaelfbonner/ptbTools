ptbDoubleNull

Double null trial

Syntax

ptbDoubleNull

Description

ptbDoubleNull waits for two 't' inputs or, if in debugging mode, two trial durations and asigns values to the following vairiables:

  • thisTrialOnset - time stamp of first 't' or first trial start in debugging mode
  • thisRelativeTrialOnset - thisTrialOnset time relative to firstTrialOnset
  • thisIsiOnset - time stamp of second 't' or second trial start in debugging mode

The following variables should be defined in the script that calls ptbDoubleNull:

  • isDebuggingRun - logical indicating whether this is a debugging run
  • isTimeZero - logical indicating whether current trial is the timeZero
  • timeZero - zero time stamp (should be an empty array if not defined yet)
  • cTrialDuration - trial duration to use during debuggin

Example

ptbDoubleNull;

Double null event

% Parameters for ptbWaitForT
WaitParameters.isDebuggingRun = isDebuggingRun;
WaitParameters.isTimeZero = isTimeZero;
WaitParameters.timeZero = timeZero;
WaitParameters.cTrialDuration = cTrialDuration;

% First null
WaitOutput = ptbWaitForT(WaitParameters);
thisTrialOnset = WaitOutput.thisTrialOnset;
thisRelativeTrialOnset = WaitOutput.thisRelativeTrialOnset;

% Second null (one trial ISI)
WaitOutput = ptbWaitForT(WaitParameters);
thisIsiOnset = WaitOutput.thisTrialOnset;
